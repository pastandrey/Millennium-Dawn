# Project Architecture

This is a Hearts of Iron IV mod, Millennium Dawn.

- `common` contains the vast majority of scripts for the mod
- `events` contains all of the events referenced in the mod
- `history` contains all of the histories for the mod such as units, countries, or states
- `docs/dev-resources/` contains all of the specific modifiers and general code stylization for the mod
- `tools` contains python scripts that can be used for standardizing or doing other work

## Coding Standards

- `code-stylization-guide` can be found in the `docs` directory and contains all of our code style.
